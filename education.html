EDUCATION

2:1 - Second Class Honours

After graduating University, I started work in Digital Marketing roles working closely with external web developers and took a great interest into the different coding languages used, along with the initial process from UX/UI design to website production. Realising this was a keen interest of mine, I enrolled into a Full Stack Web Development Diploma course at Code Institute for Software Development. The Diploma in Software Development is credit-rated by Edinburgh Napier University at level 8 on the Scottish Credit & Qualifications Framework (SCQF) with 60 SCQF credit points. Level 8 on the SCQF is equivalent to level 5 on the European Qualifications Framework (EQF). The course took a year to complete and enabled me to develop skills in main coding languages HMTL, CSS, JavaScript, Python and Django.

Along with main coding languages, the course also expanded on frameworks such as Bootstrap and Materialize, Application Program Interfaces (API's), jQuery, Flask, database programs such as MongoDB Atlas and relational database MySQL, deployment through GitHub pages (switching to PostgreSQL where applicable) and Heroku, Stripe payment system and Amazon Web Services.

I completed four milestone projects with Code Institute which were all built through GitHub using Gitpod. I used version control through Git to modify the repositories to produce a step by step log during development. Before starting the milestone development, I started with UX design by mapping out the 5 planes method of user experience and essentially created a business plan of customer and company objectives and requirements. I created wireframes for each project through using either Adobe Creative Suit programs or Balsamiq which assisted in the website development structure.
